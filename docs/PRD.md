# Product Requirements Document (PRD)

## Project Title
**Outfit of the Day (OOTD)**

## Brief Description
The Outfit of the Day application helps users generate and choose outfits for any occasion based on their existing wardrobe. By uploading pictures of clothing items, users receive personalized outfit suggestions that consider factors like weather, event type, and location.

## Technical Architecture
- **Frontend:** React (Vite) PWA located in `/client`
- **Backend:** Express API in `/api`
- **Database:** Supabase (SQL storage, authentication)
- **Version Control:** Git & GitHub
- **Hosting:** TBD (local for now)
- **Constraints:** Must support mobile-first UI and basic offline use.

## Core Features
- Upload clothing photos and categorize wardrobe.
- Filter outfits by weather or event.
- Save favorite outfits and collections.
- View community outfit feed.
- AI-influenced suggestions based on ratings.

## Target Users
Busy students, professionals, and fashion enthusiasts who want quick, personalized outfit ideas.

## Success Metrics
- Functional upload, categorization, and filtering.
- Working favorites system.
- Minimum three connected users for social feed prototype.
